# UFIS Web App

Investigation-first frontend for the UFIS backend.

## Stack

- React
- TypeScript
- Vite
- TanStack Router
- TanStack Query
- Zod
- React Flow

## Scripts

```bash
npm install
npm run dev
```

By default the Vite dev server proxies `/api/*` requests to `http://localhost:8080`.

## Current Scope

The initial scaffold includes:

- landing and search flows
- security workspace
- legal entity workspace
- corporate action detail
- security audit comparison

Corporate action creation and admin tooling remain deferred.
